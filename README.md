# go-asteroids

A Go implementation of the classic arcade game Asteroids, using Raylib-go for graphics rendering.  

In this space shooter, players must destroy asteroids while avoiding collisions in a full-screen environment.

![Asteroids Game Demo](assets/demo.gif)

## Start the game

```sh
make
```

or

```sh
go run .
```

## Controls

+ Arrow keys: Rotate and thrust the spaceship
+ Spacebar: Fire shots
+ P: Pause the game
+ R: Restart after game over
+ Esc: Exit the game
